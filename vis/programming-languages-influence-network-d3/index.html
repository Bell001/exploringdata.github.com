Programming Languages Influence Network D3

A network graph showing the connections of programming languages based on their influences. Please see the interactive programming language network for more information.

The graph is created with data from the Freebase Programming Language collection and is rendered with the JavaScript library D3 using a Force Atlas layout.

September 04, 2012 by Ramiro Gómez (@yaph).
